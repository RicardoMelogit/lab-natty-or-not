# Prompts Utilizados

## Prompt principal

Você é um Analista de Dados Júnior.

A seguir está um dataset de vendas fictício em formato CSV.
Analise os dados e apresente:

- Principais insights
- Tendências observadas
- Recomendações de negócio

Explique de forma clara, objetiva e profissional, como se estivesse
apresentando os resultados para um gestor.
